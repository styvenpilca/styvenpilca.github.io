Megamenu - Grupo 04

Megamenú

El Megamenú es una evolución de los menús desplegables (dropdowns) tradicionales. Mientras que un dropdown simple solo muestra una lista de enlaces, un megamenú permite mostrar una gran cantidad de información de forma organizada, amigable y fácil de usar.

Su principal ventaja es mejorar la usabilidad del sitio web, ya que disminuye la cantidad de clics necesarios para que el usuario llegue a cualquier parte del sitio. Es muy utilizado en sitios grandes o complejos, como tiendas de e-commerce o portales de noticias.

Características Clave
  • Multi-columna: Generalmente se diseñan en varias columnas, lo que evita el scroll vertical excesivo.
  • Contenido Enriquecido: Pueden contener elementos de conversión (formularios de registro, suscripción), imágenes de productos, videos y más.
  • Mejora la Conversión: Al poder mostrar imágenes, permite destacar productos o categorías clave.
  • Organización: La tipografía y la agrupación de opciones son cruciales para facilitar al usuario el escaneo visual de las opciones.

Ejemplo 1: Dropdown Básico
Descripción

Se agrega un menú desplegable simple. Se añade la clase .dropdown al <li> y el data-toggle="dropdown" al <a>. El menú es un <div class="dropdown-menu">.

Código
<nav class="navbar navbar-expand-lg navbar-light bg-light">
    ...
    <div class="collapse navbar-collapse" id="navEx2">
        <ul class="navbar-nav mr-auto">
            <!-- Este es el Dropdown -->
            <li class="nav-item dropdown">
                <a class="nav-link dropdown-toggle" href="#" id="navbarDropdown" role="button" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">
                    Cursos
                </a>
                <div class="dropdown-menu" aria-labelledby="navbarDropdown">
                    <a class="dropdown-item" href="#">Enlace A</a>
                    <a class="dropdown-item" href="#">Enlace B</a>
                    <a class="dropdown-item" href="#">Enlace C</a>
                </div>
            </li>
        </ul>
        ...
    </div>
</nav>

Ejemplo 2: Megamenú con Grid
Descripción

Este ejemplo aplica las clases .mega-dropdown y .mega-dropdown-menu. A diferencia del Ejemplo 1, dentro del menú usamos la cuadrícula (Grid) de Bootstrap (.row y .col-sm-4) para mostrar 3 columnas de texto simple.

Código
<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
    ...
    <div class="collapse navbar-collapse" id="navEx3">
        <ul class="navbar-nav mr-auto">
            
            <li class="nav-item dropdown mega-dropdown">
                <a class="nav-link dropdown-toggle" href="#" data-toggle="dropdown">
                    Cursos
                </a>
                
                <!-- El menú ahora contiene una fila (row) -->
                <div class="dropdown-menu mega-dropdown-menu">
                    <div class="row">
                        <!-- Columna 1 de 3 -->
                        <div class="col-sm-4">
                            <p class="text-dark small">Lorem ipsum dolor sit amet, consectetur adipisicing elit. Aut assumenda, placeat...</p>
                        </div>
                        <!-- Columna 2 de 3 -->
                        <div class="col-sm-4">
                            <p class="text-dark small">Officia ullam nobis quisquam quae cumque minima, molestias. Non porro perspiciatis...</p>
                        </div>
                        <!-- Columna 3 de 3 -->
                        <div class="col-sm-4">
                            <p class="text-dark small">Explicabo ipsa facere cum quisquam soluta ea accusamus incidunt sit amet nisi...</p>
                        </div>
                    </div>
                </div>
            </li>

        </ul>
    </div>
</nav>

Ejemplo 3: Megamenú Completo
Descripción del Código

Se añaden las 3 columnas con contenido enriquecido: un carrusel, una lista de enlaces (list-group) y una descripción con imagen.

Código
<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
    <a class="navbar-brand" href="#">Devcode.la</a>
    <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#megamenu-exec">
        <span class="navbar-toggler-icon"></span>
    </button>

        <ul class="navbar-nav mr-auto">

                    Cursos Profesionales
                </a>
                
                <div class="dropdown-menu mega-dropdown-menu">
                    <div class="row">
                        
                        <!-- Columna 1: Carrusel -->
                        <div class="col-sm-4">
                            <div id="mislider" class="carousel slide" data-ride="carousel">
                                <div class="carousel-inner">
                                    <div class="carousel-item active">
